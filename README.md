To build:

mvn clean package

To run:

java -jar target\portfolioweightcalculator-0.0.1-SNAPSHOT.jar <input file>

Example usage:

java -jar target\portfolioweightcalculator-0.0.1-SNAPSHOT.jar testinput.txt
